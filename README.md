# claude-registry

A curated collection of Claude Code plugins.

## Usage

Add this marketplace to Claude Code:

```bash
claude plugin marketplace add robertbagge/claude-registry
```

Then install plugins from it:

```bash
claude plugin install roadhouse@claude-registry
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [roadhouse](https://github.com/robertbagge/claude-roadhouse-plugin) | Iterative code improvement loop using `/proud` and `/exquisite` introspection |
